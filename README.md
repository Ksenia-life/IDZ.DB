# Итоговый проект по базам данных

## О проекте
Проектирование и реализация базы данных для библиотечной системы на PostgreSQL.

## Структура базы данных
База содержит 8 таблиц:
- Читатели (readers)
- Книги (books)
- Авторы (authors)
- Жанры (genres)
- Займы (loans)
- Штрафы (fines)
- Связующие таблицы (book_authors, book_genres)
